Hướng dẫn chạy source:

- Vào thư mục Frontend/clients/pages, mở file home_page.html lên và trải nghiêm.

- Nếu muốn thử nghiệm với server Backend, hãy mở terminal chứa source-code (Không phải thư mục Backend), gõ npm run server và truy cập vào localhost:3000/book để kiểm tra.

Để tiết kiệm thời gian, Web App này đã được deploy trên tên miền mangocomic.io.vn. Bạn hãy truy cập và trải nghiệm.
